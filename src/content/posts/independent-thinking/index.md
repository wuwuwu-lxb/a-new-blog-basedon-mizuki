# 你所处的世界和你生活的世界一样吗？

我们确确实实的生活在同一个世界，但哪怕是处于同样客观条件下的两个人，由于个体意识的不同，导致两人所生活的世界其实并不相同，也就是主观上的体验不同，看法不同，思想不同，而在行动来源于自驱的情况下，接下来的行动会截然不同，而这些行动就是自己和外在世界的雕刻刀，进一步改变自我意识和客体环境。所以如果你也进入了这样一个循环，该怎样破局。  
# 信息的接收
我们究其根本，我们接收的信息以及处理信息的方式影响了我们的输出的情绪和行动。人脑的记忆率只有0.00036%，人每秒从五官中获取的信息高达1100万条，但实际记住的只有40条。这表明的我们采集的信息虽然很多，但真正能被我们记下来的其实只有很小一部分。而这种记忆模式由两部分组成，一方面，是你的潜意识，另一方面，是你的注意力。假如事先告诉你，看一张图的目的是记住图中人的发色，那就会忽略掉其他成分，譬如衣服颜色。但通常情况下，我们只是依靠着潜意识，没有人是没日没夜的去带着目的去采集信息，重要的是，我们的注意力是有限的。  
## 信息的处理
收集信息之后，便是靠我们的思维模式进行处理，人一天大概会思考，动念6200次，大概15秒钟一个想法。但我们实际感觉上并没有那么的累，因为被早就形成的思维模式操控了，它就像是我们在厚厚雪堆里走出来的路，路的尽头就是输出。而这里同样有着巨大的影响，对于同样的信息，我们处理的手段不同导致的输出甚至影响比信息收集更大。不过这里不加以证明了。也不拓展。   
  
## 简单的概括：
  
>注意力（聚焦） → 思维模式（解读） → 情绪与执行力（内化输出） → 行动（外化输出） → 改变客观世界（反馈） → 影响注意力（新的聚焦）…    

>但实际上，我们大部分时间是这样的：
 随机采样→ 固定思维模式解读→ 执行力随机产生，行动受情绪控制 → 改变客观世界 → 影响样本   

## 调整
所以，我们有两个方面需要调整，
- 一个是注意力方面
- 一个是思维模式方面。   
关于这两个，我会放另外课题。  
## 尾语
除此之外，我想说：我们生活在的世界其实是我们的真正的一种内心渴求和根本意图所构建出来的，你关心什么，什么就会作为“生活的世界”向你显现。这种渴求，是复杂的，是客观与主观共同影响的，是神奇的，也是人最根本的动力所在。当一个人失去了渴求，意义所搭建出的生活的世界崩塌了，动力也不复存在。我们努力构建自己生活的世界，也仅仅是在客体无意义的世界中找寻我存在在此的意义。

- 可是，你要知道，生活的世界不是被“塑造”的，而是被“召唤”出来的
